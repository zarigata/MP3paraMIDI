## [0.1.0] - 2025-01-01

### Added
- Initial release of MP3paraMIDI.
- Cross-platform GUI application (Windows, macOS, Linux).
- Audio to MIDI conversion with monophonic pitch detection.
- AI-powered polyphonic detection via Basic-Pitch.
- Source separation with Demucs.
- Note quantization with configurable grid sizes.
- Automatic tempo detection and enhanced filtering.
- MIDI preview playback with pygame.
- Comprehensive settings dialog with persistent preferences.
- Multi-track MIDI generation and improved velocity detection.

### Technical
- PyInstaller-based distribution pipeline.
- Pre-built binaries for Windows, macOS, and Linux.
- GitHub Actions CI/CD workflow for builds and releases.
